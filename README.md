# German Flashcards

A minimalist vintage-style flashcard web app for learning German vocabulary.

## Files

- `index.html` - Main flashcard application (don't edit this)
- `vocab.txt` - Your vocabulary list (edit this to add/modify words)

## How to Update Vocabulary

Simply edit `vocab.txt` using any text editor. The format is super simple:

```
English word | German word | Optional hint
```

### Rules:
- Separate English and German with a pipe character: `|`
- One word/phrase per line
- Lines starting with `//` are comments (ignored)
- Empty lines are ignored
- Third column (hint) is optional

### Example:
```
// Greetings
Hello | Hallo
Goodbye | Tsch√ºss | informal
Thank you | Danke

// Food
Apple | der Apfel | masculine noun
```

That's it! Just save the file and refresh your webpage.

## How to Upload to GitHub

1. **Create a new repository** on GitHub
2. **Name it** (e.g., `german-flashcards`)
3. **Upload both files:**
   - `index.html`
   - `vocab.txt`
4. **Enable GitHub Pages:**
   - Go to Settings ‚Üí Pages
   - Source: Select "main" branch
   - Click Save
5. **Access your site** at:
   - `https://[your-username].github.io/german-flashcards`

## Updating Vocabulary After Publishing

To add more words after you've published to GitHub:

1. Go to your repository on GitHub
2. Click on `vocab.txt`
3. Click the pencil icon (Edit)
4. Add your new words in the same format
5. Commit changes
6. Your website will update automatically!
